### Installation  

1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/EcoHack-Frontend.git
   ```  

2. Navigate to the project directory:  
   ```bash
   cd EcoHack-Frontend
   ```  

3. Install dependencies:  
   ```bash
   npm install
   # or
   yarn install
   ```  

4. Start the development server:  
   ```bash
   npm start
   # or
   yarn start
   ```
